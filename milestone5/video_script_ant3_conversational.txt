Duration: 2.5 minutes
Direct, Technical Explanation

[0:00-0:10] PROBLEM SETUP
───────────────────────────────────────────────────────────────
Multi-robot coverage problem: 3 robots, 6x6 grid, 30 free cells, 
6 obstacles. Goal: cover all free cells efficiently.

[0:10-0:30] ACO OVERVIEW
───────────────────────────────────────────────────────────────
Ant Colony Optimization uses virtual ants to explore solutions. 
Each ant creates a solution in two steps:

1. DARP (Divide Areas based on Robot Positions): Assigns cells 
   to robots using probability based on pheromone and desirability.

2. UF-STC (Uniform Coverage Spanning Tree Construction): Builds 
   coverage paths for each robot.

UF-STC works in two phases:
- Phase 1: Build spanning tree using BFS (Breadth-First Search) 
  over assigned cells, connecting all cells in a tree structure.
- Phase 2: Generate coverage path using DFS (Depth-First Search) 
  preorder traversal, visiting each cell at least once while 
  maintaining path continuity.

[0:30-0:55] DESIRABILITY FUNCTION
───────────────────────────────────────────────────────────────
Desirability η determines cell-robot assignment quality.

Formula: η = 1 / (dist + γ|L_r - L̄|)

Where:
- dist: Manhattan distance from robot start to cell
- L_r: Current robot path length
- L̄: Average path length across all robots
- γ (gamma): Workload balance weight (set to 1.0)

High desirability = cell is close AND assignment balances workload.

[0:55-1:15] PROBABILITY CALCULATION
───────────────────────────────────────────────────────────────
Ants assign cells to robots using probability:

P(r|i) ∝ τ[r][i]^α × η[r][i]^β

Where:
- τ[r][i]: Pheromone level for robot r and cell i
- η[r][i]: Desirability for robot r and cell i
- α (alpha): Pheromone importance (set to 1.0)
- β (beta): Heuristic importance (set to 1.0)

High α = trust past experience more. High β = prefer desirability.

[1:15-1:40] PHEROMONE UPDATE
───────────────────────────────────────────────────────────────
Pheromone update happens in two steps:

Step 1 - Evaporation:
τ[r][i] = τ[r][i] × (1 - ρ)

Where ρ (rho) is evaporation rate (set to 0.5). Old information fades.

Step 2 - Deposit (Ant Quantity Model):
τ[r][i] = τ[r][i] + η[r][i]

Each ant deposits pheromone equal to desirability value. Good cells 
get more pheromone, guiding future ants.

[1:40-2:00] ITERATION PROCESS
───────────────────────────────────────────────────────────────
Algorithm runs 50 iterations. Each iteration:
1. 15 ants construct solutions
2. Calculate desirability for each cell-robot pair
3. Use probability to assign cells (DARP)
4. Build paths using UF-STC
5. Evaporate pheromone: τ = τ × (1-ρ)
6. Deposit pheromone: τ = τ + η
7. Track best solution

[2:00-2:25] CASE STUDY 2 RESULTS
───────────────────────────────────────────────────────────────
Ran algorithm 4 times for reliability testing.

Results:
- Coverage: 100% (30/30 free cells)
- Workload imbalance (F2): 12.00
- Runtime: 7.3 seconds average
- Best solution path lengths: Robot 0: 11 cells, Robot 1: 21 cells, 
  Robot 2: 19 cells
- All paths feasible: no obstacles, no jumps

[2:25-2:30] CONCLUSION
───────────────────────────────────────────────────────────────
ACO successfully solves multi-robot coverage using pheromone trails, 
multi-objective desirability, and probability-based assignment. 
Achieves 100% coverage with balanced workload in 7.3 seconds.

[END]

KEY FORMULAS:
• Desirability: η = 1 / (dist + γ|L_r - L̄|)
• Probability: P ∝ τ^α × η^β
• Evaporation: τ = τ × (1-ρ)
• Deposit: τ = τ + η

PARAMETERS:
• α = 1.0 (pheromone weight)
• β = 1.0 (heuristic weight)
• γ = 1.0 (workload balance weight)
• ρ = 0.5 (evaporation rate)
• Iterations: 50
• Ants per iteration: 15
